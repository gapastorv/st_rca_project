En la primera versión (1.0), se tiene lo siguiente:
- Una clase de extracci;on de tablas usando tabula, DataExtractor.
- Una clase para acceder a la base de datos, Firebase.
- Una clase para realizar el chequeo constante del correo con imaplib, MailChecker.
